## 2 хэмжээст тэмдэгтэн хүснэгт
Дараах жишээг зурагт үзүүлэв.
```c
int main(){
  int A[3][3] = {{0,1,2},{3,4,5},{6,7,8}};
  for(int i=0;i<3;i++){ //Mur
    for(int j=0;j<3;j++){ //Bagana
      cout<<A[i][j]<<" ";
    }
    cout<<'\n';
  }
  return 0;
}
```
